Welcome to Game of Set!

Game Rules

The object of the game is to identify a 'Set' of three cards from 12 cards laid out on the table. Each card has a variation of the following four features:

1) COLOR: Each card is red, green, or blue.

2) SYMBOL: Each card contains ovals, squiggles, or diamonds.

3) NUMBER: Each card has one, two, or three symbols.

4) SHADING: Each card is solid, open, or striped.

A 'Set' consists of three cards in which each feature is EITHER the same on each card OR is different on each card. That is to say, any feature in the 'Set' of three cards is either common to all three cards or is different on each card.

One Player Game:

Click the "Single Player" to start playing a singleplayer game. You must press the "Player 1" button to buzz-in before trying to check a set using the "Check" button.

Two Player Game:

Click the "Two Players" to start playing a two player game. Each player must press their respective "Player 1" and "Player 2" buttons to buzz-in before trying to check a set using the "Check" button.
